## Steps To Install

1. Open VS Code
2. Click the Extensions icon on the sidebar or press `Ctrl+Shift+X` (Windows/Linux) or `Cmd+Shift+X` (macOS)
3. In the search bar, type ‚ÄúSaarthi‚Äù.  Select <SaarthiIcon/> Saarthi by godspeed and click **Install**
4. Reload VS Code if prompted
5. Click on Saarthi icon, you'll then see Saarthi welcome screen.
5. Here, Select LLM Provider and enter API Key. For help on connecting your API provider [ssCheck this](/docs/saarthi/getting-started/connecting-api-provider)
- To use Free Google API Key. Get it from [Google AI Studio](https://aistudio.google.com/app/apikey)

## Troubleshooting

*VS Code Output panel showing Saarthi logs for troubleshooting*

**Extension Not Visible**
* Restart VS Code
* Verify Saarthi is listed and enabled in Extensions
* Try disabling and re-enabling
* Check Output panel for errors (View ‚Üí Output, select "Saarthi")

**Installation Problems**
* Ensure stable internet connection
* Reload VS Code in case installation takes more than 10 times.
